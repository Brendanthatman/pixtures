Concorde vs TU-144

rate

This image was taken at the Technik Museum in Sinsheim, Germany. This place is a very special place, with this being a very special picture, because it is the only place in the world where one can see the TU-144 aircraft outside of Russia. That being said, this is also the only place one can see both the Concorde and TU-144 together on display, an enter both. A truly special memory to be able to admire and enter both of these birds.
